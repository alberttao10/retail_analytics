
## Key Insights

### 🛒 Store Performance
- Walmart generates the highest total revenue
- Dollar stores show high transaction count but lower average spend

### 📦 Product Trends
- Snacks category dominates total sales
- Health & Wellness products have higher average price

### 👤 Customer Behavior
- A subset of users are repeat buyers
- High variance in purchase frequency

### 📁 Repository Details
This project analyzes retail transaction data to answer:
- Which stores drive the most revenue?
- Which product categories perform best?
- What are customer purchase patterns?

Tech used:
- PostgreSQL
- DataGrip

Data Architecture:
- bronze layer: raw csv files manually imported using UI
- silver layer: cleaned up, query-ready data
- gold layer: further cleaned, business insight ready queries
